# Practica-6-Ejecutar-servidor-web-Nginx-con-Docker

Este proyecto consiste en la creación y despliegue de un servidor web Nginx utilizando Docker Compose. Se configuró un contenedor Nginx para servir una página web básica alojada en un directorio local (src) mediante el uso de volúmenes. El objetivo de la práctica es familiarizarse con la creación de contenedores, la configuración de Docker Compose y la publicación de proyectos mediante GitHub.
